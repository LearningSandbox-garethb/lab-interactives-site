Standalone Interactive Creation Tool

  1. Download the resources needed for all interactives: lab-interactive.tar.gz
  2. Unzip the file above which should give you a folder 'lab-interactive'
  3. Put the path to the interactive after the '#' in the URL above and reload this page
  4. Copy the text below into a text editor and save as an html file inside the 'lab-interactive' folder created by the file
  5. Open this new file in a browser, you should see a file:// URL and the interactive should load.
  6. Disconnect your network, clear your browser cache and try loading the file again to make sure it has everything it needs.

NOTE: If the interactive or model has resources it references (like images) you will need to track those down and put them in the appropriate place.
